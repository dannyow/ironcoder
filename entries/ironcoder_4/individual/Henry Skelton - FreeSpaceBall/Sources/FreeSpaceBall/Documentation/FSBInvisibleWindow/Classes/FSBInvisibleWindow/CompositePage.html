FSBInvisibleWindow


Abstract

This object is an invisible NSWindow.

Discussion

This is exactly like and NSWindow, but the init method causes it to be invisible.

Methods

initWithContentRect

- (id)initWithContentRect:(NSRect)contentRect styleMask:(unsigned int)aStyle backing:(NSBackingStoreType)bufferingType defer:(BOOL)flag;

This is like a standard NSWindow init, but it creates an invisible window

Result: An invisible NSWindow is created
