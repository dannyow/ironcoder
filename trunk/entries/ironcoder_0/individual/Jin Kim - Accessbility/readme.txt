This is a little application where you can shoot the beads (F1) to your favorite application then the number of beads an application carries goes up.
Just like how you would throw beads to your favorite flasher in Mardi Gras.

Instruction:
Push F1 function key (fn+F1 is you are on a laptop) while your cursor is over an application you what to shoot the beads to.
