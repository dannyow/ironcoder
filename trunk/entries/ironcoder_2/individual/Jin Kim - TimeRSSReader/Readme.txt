TimeRSSReader is an RSS Reader for Time.com RSS feeds.
It creates a Time magazine like cover page using an article's title.
It searches Flickr photos using selected article's title and chooses a random photo from the search results to be used as a cover art.

Instructions:
- Choose from different RSS feeds.
- Select article or click "Random" button to randomly choose article.
- You can choose "Read Article" button which will take you to the article in Time.com website or "Save as..." button will save the current cover page as PDF file.
